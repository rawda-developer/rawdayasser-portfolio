My-Reads

Dec 2021 - Dec 2021

View on GitHub

MyReads is a front-end Web App built in React js.

  • It allows users to categorize books into 3 categories, called shelves (currently reading, want to read, and read).
  • The application interacts with an API to get books names, authors, and images.
  • It's a Single-page Application with 2 routes for Home and Search pages.
  • The project consists of 7 reusable React Components for building the UI of Home, and Search pages.
  • This project was a part of React Nanodegree Program at Udacity.
